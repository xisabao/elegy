Elegy is basically Isabelle trying to make a better Haiku for her Intro to Comp Sci final project but failing miserably. It's a parody, guys.

Quick note: The great thing about this website is that administrator access is required to change your settings, so even if you do create an account, an administrator will have to set your permissions and settings. HA! Have fun messing up my project now!

Fun fact: Elegy is named 'Elegy' because originally the project was named 'Sonnet' so I could call it Sonnet Project in honor of our Freshman English Sonnet Projects, but I messed up my database migrations and SQL table joins, so I had to start a new project. In my rage and frustration, I Googled for forms of poems. I scrolled down and the first thing I saw was Elegy, which is a poem lamenting its subject's death. Perfect for a site lamenting the death of Haiku, right? (Also perfect for me, lamenting the death of my database :/)

Speaking of failures, one day, while working on this project, I managed to spend an entire morning debugging a weird error with my password hashing. Four hours later, a kind stranger informed me that it was because I forgot one pair of parentheses. Thanks, Ruby.

Looks pretty awesome right? Well, you too can do this and be as awesome as me! I used the following tools (and many hours procrastinating, do not recommend) to make this website:

  • Django, a full-stack web application framework
  • Bootstrap, a front-end CSS and JS framework
  • Heroku, for easy deployment
  • Fartscroll.js, the greatest Javascript library ever created

As well as knowledge of:

  • Python, of course. Thank you to Andrew for an awesome year of Comp Sci!
  • Javascript and JQuery, just to make this awesome modal and drive in the fact that I know Javascript too.
  • HTML and CSS, of course...
  • Just enough SQL to get by...
  • Some miscellaneous stuff like CLI, Git and Regex
  • How to Google every bug you come across
  • How to skim through ridiculous amounts of documentation
  • How not to mess up your database migrations
  • How to post dumb questions on Stack Overflow

It did take me a couple months to learn all of this, though.
